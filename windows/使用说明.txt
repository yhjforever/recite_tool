============================================================
        背诵稿生成器 · 使用说明（给第一次用的你）
============================================================

这是什么？
  一个把「课件 PPT / 电子课本」按「课程大纲」整理成"可直接背诵的
  Markdown 文档"的小工具。它调用 DeepSeek 来整理，忠实原文、按大纲
  分章、排版清爽，课件没讲到的会标〔补〕提示你核对。


【运行环境】
  · Windows 10 / 11，64 位电脑。
  · 能联网（要调用 DeepSeek；用"联网核对"还要能上网搜索）。
  · 不需要安装 Python，双击 exe 即可。


【你需要先准备】
  1. 一个你自己的 DeepSeek API Key
     去 https://platform.deepseek.com 注册 → 创建 API Key（sk- 开头）。
     注意：每人用自己的 Key，各自计费；别人不会用到你的，你也别用别人的。
  2. 一个文件夹，放这门课的资料：
     · 所有课件（PDF 最稳；本版也支持 .pptx；电子课本 PDF 也可一起放）
     · 一份"教学大纲"文件（文件名最好含"大纲"二字，便于自动识别）


【关于这个文件夹】
  · 本工具是一个文件夹（启动更快）。请把整个文件夹解压后保持完整，不要只拷出 exe。
  · 运行的就是文件夹里的「背诵稿生成器.exe」；旁边的 _internal 等文件是它的运行库，别删。

【第一次怎么用】（全程点鼠标，不用敲命令）
  1) 双击文件夹里的「背诵稿生成器.exe」。
     · 若弹出蓝色窗口"Windows 已保护你的电脑"：点【更多信息】→【仍要运行】。
     · 若杀毒软件报警：这是 PyInstaller 打包的常见误报，选择"允许/信任"即可。
  2) 第一次会让你填 DeepSeek API Key —— 填一次就存在本机，以后不再问。
  3) 点【选择…】挑你刚才那个资料文件夹。
  4) 填"学科名称"（留空也行，会自动判断）；
     "课程说明/整理偏好"可选填，比如：细节要求高、名词第一次出现时后面加英文 等。
  5) 点【审计资料】——它会自动判断学科、把每个课件对到大纲各章，
     下面表格会列出章节（含状态、素材文件、大纲字数）。
  6) 在表格里：单击某行最左边的【☑】列勾选要做的章节（可用【全选】）；
     单击行本身，右侧"章节详情"会显示该章的素材、大纲摘录、输出与待核情况。
     选好后点【生成所选章节】。生成的 .md 会保存在"你的资料文件夹 / output"里。
     按钮会按当前状态自动灰显/可用（没设 Key、没审计时相应按钮不可点，并有提示）。
  7) （可选）点【④ 联网核对】：对课件没讲到的点联网查权威资料补充，并附真实网址。


【输出在哪 / 怎么看】
  · 在你资料文件夹下的 output 文件夹里，每章一个 .md 文件。
  · .md 用记事本能看；想看排版效果可用 Typora、VS Code、或网页版 Markdown 工具打开。
  · 点界面【打开输出文件夹】可直接跳过去。


【常见问题】
  · 启动慢几秒？正常，单文件 exe 每次要先解压。
  · 想重做某章？勾上【覆盖重做】再点【③ 生成所选】。
  · 某章带〔补〕？表示课件/课本里没有、是补充的，可能不准，请自己核对，
    或点【④ 联网核对】换成带真实网址的版本。
  · 资料最好用 PDF；.pptx 本版已支持，但个别复杂排版仍以 PDF 最稳。


【一句话流程】
  填自己的 Key → 选资料文件夹 → ①审计 → 勾章节 → ③生成 →（可选④联网核对）
  → 去 output 文件夹拿 .md。
