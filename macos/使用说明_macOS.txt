============================================================
      背诵稿生成器 · 使用说明（macOS · 给第一次用的你）
============================================================

这是什么？
  一个把「课件 PPT / 电子课本」按「课程大纲」整理成“可直接背诵的
  Markdown 文档”的小工具。它调用 DeepSeek 来整理，忠实原文、按大纲
  分章、排版清爽，课件没讲到的会标〔补〕提示你核对。


【运行环境】
  · macOS 11 (Big Sur) 及以上，Apple 芯片(M 系列)或 Intel 均可。
  · 能联网（要调用 DeepSeek；用“联网核对”还要能上网搜索）。
  · 只需装一次 Python 3.10+（推荐 python.org 官方安装包，自带 Tkinter）。
    其余依赖由程序首次启动时自动安装，全程不用打开“终端”。
    （若收到的是已打包的 .app，则连 Python 都不用装，见“方式 B”。）


【你需要先准备】
  1. 一个你自己的 DeepSeek API Key
     去 https://platform.deepseek.com 注册 → 创建 API Key（sk- 开头）。
     注意：每人用自己的 Key，各自计费；别人不会用到你的，你也别用别人的。
  2. 一个文件夹，放这门课的资料：
     · 所有课件（PDF 最稳；也支持 .pptx；电子课本 PDF 也可一起放）
     · 一份“教学大纲”文件（文件名最好含“大纲”二字，便于自动识别）


【方式 A · 收到 zip 压缩包：解压即点（推荐）】（全程点鼠标）
  1) 把「recite_tool_mac.zip」拷到 Mac，放到“下载/文稿/桌面”等普通文件夹。
  2) 双击解压，得到「recite_tool_mac」文件夹。
  3) 打开文件夹，双击「启动.command」。
     · 首次若提示“无法验证开发者/来自互联网”：右键点「启动.command」→
       「打开」→ 弹窗里再点「打开」。（仅第一次需要这一下，之后直接双击。）
     · 首次会自动安装依赖（约 1 分钟，仅此一次），随后自动打开窗口。
     · 想要桌面图标：双击一次「创建桌面快捷方式.command」。
  4) 第一次会让你填 DeepSeek API Key —— 填一次就存在本机，以后不再问。
  5) 点【选择…】挑你那个资料文件夹。
  6) 填“学科名称”（留空也行，会自动判断）；
     “课程说明/整理偏好”可选填，比如：细节要求高、名词后加英文 等。
  7) 点【① 审计资料】——自动判断学科、把课件对到大纲各章，下面列出章节。
  8) 勾选要做的章节（单击逐个勾，或【全选】）→ 点【③ 生成所选】。
     生成的 .md 会保存在“你的资料文件夹 / output”里。
  9) （可选）点【④ 联网核对】：对课件没讲到的点联网查权威资料并附真实网址。


【方式 B · 收到 .app：双击即用（无需装 Python）】
  1) 把「背诵稿生成器.app」放到可写文件夹（下载/文稿/桌面）——程序会把
     你的设置/Key 存到 .app 同级目录。
  2) 双击打开。首次若提示“无法验证开发者/来自互联网”：
     · 右键点 .app →「打开」→ 在弹窗里再点「打开」；或
     · 到「系统设置 → 隐私与安全性」→ 点「仍要打开」。
  3) 之后与方式 A 第 4 步起相同：填 Key → 选资料文件夹 → 审计 → 生成。


【输出在哪 / 怎么看】
  · 在你资料文件夹下的 output 文件夹里，每章一个 .md 文件。
  · .md 用“文本编辑”能看；想看排版可用 Typora、VS Code、或网页版 Markdown 工具。
  · 点界面【打开输出文件夹】可直接跳过去（访达打开）。


【常见问题】
  · 双击「启动.command」提示“无法验证开发者”？右键点它→「打开」一次即可。
    （小贴士：用 U 盘 / 隔空投送 / 局域网把 zip 传过来的，通常连这一下都不用。）
  · 第一次启动卡在“正在安装依赖”？属正常，约 1 分钟，仅此一次；完成后自动开窗。
  · 想重做某章？勾上【覆盖重做】再点【③ 生成所选】。
  · 某章带〔补〕？表示课件/课本里没有、是补充的，可能不准，请自己核对，
    或点【④ 联网核对】换成带真实网址的版本。
  · 提示缺少 Tkinter？请改用 python.org 的 Python，或 brew install python-tk。
  · 资料最好用 PDF；.pptx 已支持，但个别复杂排版仍以 PDF 最稳。


【一句话流程】
  填自己的 Key → 选资料文件夹 → ①审计 → 勾章节 → ③生成 →（可选④联网核对）
  → 去 output 文件夹拿 .md。
